# 色块高度与花纹叠加设计说明

## 目标
- 将色块统一提升到 120–160px 的纵向比例，增强层次与可读性
- 为色块背景叠加低干扰的简约花纹（几何线条/点阵/网格），并提供开关与透明度参数

## 实现概览
- 高度变量：`--block-height` 默认 140px，移动端（<=720px）为 124px
- 花纹素材：`pages/assets/patterns/` 下的 `pattern-dots.svg`、`pattern-lines.svg`、`pattern-grid.svg`
- 花纹变量：
  - `--pattern-opacity` 默认 0.18（建议范围 0.10–0.25）
  - `--pattern-size` 默认 28px
  - `--pattern-image` 默认 `var(--pattern-dots)`，可改为 `var(--pattern-lines)` 或 `var(--pattern-grid)`
- 开关：在 `<body>` 设置 `data-pattern="on" | "off"`；`off` 时花纹透明度为 0

## 关键样式
### 白条猪子页面
- 统计卡片 `.stat-card`：
  - 高度：`min-height: var(--block-height);`
  - 花纹层：伪元素 `::before` 叠加 `background-image: var(--pattern-image)` 与 `opacity: var(--pattern-opacity)`
  - 混合模式 `mix-blend-mode: soft-light`，保证主内容不受干扰

### 采购分析仪表板子页面
- 左侧菜单 `.menu-item`：
  - 高度：统一提升到 `min-height: var(--block-height)`
  - 花纹层：伪元素 `::after` 同样叠加花纹
  - 断点：在 720px 以下将侧边菜单切换为两列网格，提升空间利用率

## 使用建议
- 花纹颜色始终采用白色叠加并通过透明度控制对比度不超过主色的 60%
- 若页面中存在大量文字内容的卡片，建议将 `--pattern-opacity` 降至 `0.12` 左右
- 避免在同一屏中同时使用“高饱和底色 + 高透明度花纹 + 大面积文字”，会显得噪点过强

## 交付内容清单
- 样式代码：已更新于 `pages/baitiao-pig-system/styles.css` 与 `pages/procurement-dashboard/styles.css`
- 素材：`pages/assets/patterns/*.svg`
- 交互参数：`data-pattern` 开关、`--pattern-opacity` 透明度、`--pattern-size` 尺寸、`--pattern-image` 花纹类型

## 响应式说明
- 断点 1100px：白条猪页面上下区域单列排布，避免横向拥挤
- 断点 720px：块高降至 124px；采购分析左侧菜单改为 2 列网格；保证触控区域足够
