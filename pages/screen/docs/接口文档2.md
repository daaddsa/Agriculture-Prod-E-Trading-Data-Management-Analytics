## 1：批发市场数据接口

**1.接口说明**

**2.使用场景**

此接口主要用于根据批发市场搜索实时交易数据。

**3.接口调用说明**

**3.1请求说明**

  -------------- ----------------------------------------------------------------------------------------------------------------------------------------------------
  url            [http://xu9sp2bdxt3d.guyubao.com/tradeDynamicData/getTradeMarket](http://9gdbmhh9j1e1.ngrok.xiaomiqiu123.top/tradeDynamicData/abnormal?marketId=1)

  支持验证方式   无

  格式           JSON

  http请求方式   GET

  是否需要鉴权   不需要
  -------------- ----------------------------------------------------------------------------------------------------------------------------------------------------

**3.2输入参数说明**

无

**3.3请求示例**

[http://xu9sp2bdxt3d.guyubao.com/tradeDynamicData/getTradeMarket](http://9gdbmhh9j1e1.ngrok.xiaomiqiu123.top/tradeDynamicData/abnormal?marketId=1)

**3.4返回参数说明**

  ------------------ ----------------------------------------------------
     **参数说明**                          **描述**

         code                               返回码

         msg                           相应的信息提示。

         data                             返回数据集

## 2：可视化指标数据接口

**1.接口说明**

获取可视化今日交易总量、交易金额、交易均价、交易均价（不含异常交易）、货源企业数量、采购商数量、产地销售量排名、销地采购量排名、屠宰厂平均出厂价。

**2.使用场景**

此接口主要用于获取日交易总量、交易金额、交易均价、交易均价（不含异常交易）、货源企业数量、采购商数量、产地销售量排名、销地采购量排名、屠宰厂平均出厂价实时交易数据。

  格式           JSON

  http请求方式   GET

  是否需要鉴权   不需要
  -------------- ------------------------------------------------------------------------

**3.2输入参数说明**

marketId ：批发市场编号
